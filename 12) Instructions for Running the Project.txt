1) Clone the repository:

            git clone https://github.com/yourusername/real-time-climate-forecasting.git cd real-time-climate-forecasting

2) Configure the project:

            Add your API key and city name in config/config.json.

3) Compile the code: 

           make

4) Run the application:

            ./climate_forecasting

            
Run the benchmarks (optional): Open benchmark.ipynb and run the cells to view performance analysis.
